# Backend

# Installation/Setup
1. Create `.env-dev` with following setup code:

    ```
    JWT_SECRET=
    MONGODB_URI=mongodb://

    MINIO_SERVER=
    MINIO_PORT=9000
    MINIO_SSL=false
    MINIO_ACCESS_KEY=
    MINIO_SECRET_KEY=
    MINIO_SERVER=

    DEFAULT_ACL_GROUP=User
    ```
2. `npm run build`
3. `npm run debug`
